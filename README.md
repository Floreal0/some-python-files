研究生调剂结束，去向不尽人意。但相对于过去庸庸碌碌的一年，此刻却是难得的空闲时间。GitHub许久未登，此次计划重构代码，才发现原来现在还保留的第一个fork仓库已经是2019年的事情了——七年一晃而过。平心而论，我的代码能力有减无增(这是2023重构想法没有实施的部分原因)，但AI极大降低了信息搜索能力的要求，提供框架选择和具体代码修改的助力，因此，此次重构计划可行。
